## Парсер сайта [наш.дом.рф](https://наш.дом.рф/сервисы/каталог-новостроек/список-объектов/список?place=0-1&objStatus=0) на фреймворке [Scrapy](https://scrapy.org)

## Зависимости 
python = "^3.11"
poetry = "^1.6.1"

## Использование

Установка:
```sh
poetry install
```

Запустите парсер:
```sh
poetry run parser
```
